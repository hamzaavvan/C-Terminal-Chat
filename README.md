# C-Terminal-Chat
A simple chatapp coded in C using shm

## Compile program
```bash
$ cd code
$ gcc chatserver.c -o server
$ gcc chatclient.c -o client
```

## Executing Chatapp
First start the server in a separate terminal
```bash
$ ./server
```

Then start the clients in different terminal windows
```bash
$ ./client
```
